## Usage

### Use bash

```bash
$ git clone https://github.com/seecent/cms-admin.git --depth=1
$ cd cms-admin
$ npm install
$ npm start         # visit http://localhost:8000
```

### Use by docker

```bash
// dev 
$ npm run docker:dev

// build 
$ npm run docker:build


// production dev 
$ npm run docker-prod:dev

// production build 
$ npm run docker-prod:build
```
